ConvertKit API

The ConvertKit REST API enables custom integrations for developers.

API Access

  • Available: All paid plans (Free plan has limited API access)
  • Documentation: api.convertkit.com/docs
  • Rate Limit: 300 requests per minute
  • Authentication: API Secret Key

Getting Your API Key

  1. Log in to ConvertKit
  2. Go to Settings → Advanced
  3. Scroll to "API Secret" section
  4. Copy your API Secret (keep this secure!)

API Endpoints

Endpoint Method Use Case
/subscribers GET, POST List or create subscribers
/tags GET, POST List or create tags
/tags/{id}/subscribers POST Tag a subscriber
/sequences GET List email sequences
/sequences/{id}/subscribers POST Add subscriber to sequence
/forms GET List signup forms
/forms/{id}/subscribe POST Subscribe via form
/purchases POST Record a purchase

Example: Create Subscriber with Tag

import requests

API_SECRET = 'your_api_secret_here'
BASE_URL = 'https://api.convertkit.com/v3'

def create_subscriber(email, first_name, tag_id):
    """Create a subscriber and add tag in one call"""
    response = requests.post(
        f'{BASE_URL}/subscribers',
        headers={'Content-Type': 'application/json'},
        json={
            'api_secret': API_SECRET,
            'email': email,
            'first_name': first_name,
            'tags': [tag_id]
        }
    )
    return response.json()

# Example usage
result = create_subscriber(
    email='customer@example.com',
    first_name='Sarah',
    tag_id=12345  # "customer" tag ID
)
print(f"Subscriber created: {result['subscriber']['id']}")

Example: Add to Sequence via Node.js

const axios = require('axios');

const API_SECRET = 'your_api_secret_here';
const BASE_URL = 'https://api.convertkit.com/v3';

async function addToSequence(email, sequenceId) {
  try {
    const response = await axios.post(
      `${BASE_URL}/sequences/${sequenceId}/subscribers`,
      {
        api_secret: API_SECRET,
        email: email
      }
    );
    console.log('Added to sequence:', response.data);
  } catch (error) {
    console.error('Error:', error.response?.data || error.message);
  }
}

// Add customer to welcome sequence
addToSequence('new@customer.com', 67890);

Webhooks Setup

Webhooks send real-time notifications when events occur in ConvertKit.

Available Webhook Events

Event Trigger
subscriber.form_subscribe Subscriber joins via form
subscriber.sequence_subscribe Subscriber added to sequence
subscriber.tag_add Tag added to subscriber
subscriber.tag_remove Tag removed from subscriber
subscriber.purchase_add Purchase recorded
subscriber.purchase_remove Purchase removed

Setting Up Webhooks

  1. Go to Settings → Integrations → Webhooks
  2. Click "Add Webhook"
  3. Enter your endpoint URL (e.g., yourserver.com/webhooks/convertkit)
  4. Select events to track
  5. Save and test

Webhook Handler Example (Node.js/Express)

const express = require('express');
const app = express();

app.use(express.json());

app.post('/webhooks/convertkit', (req, res) => {
  const { event, subscriber, tag } = req.body;
  
  console.log(`Event: ${event}`);
  console.log(`Subscriber: ${subscriber.email}`);
  console.log(`Tag: ${tag?.name || 'N/A'}`);
  
  // Handle different events
  switch (event) {
    case 'subscriber.tag_add':
      if (tag.name === 'customer') {
        // Add to CRM, send welcome email, etc.
        syncToCRM(subscriber);
      }
      break;
    case 'subscriber.purchase_add':
      // Trigger fulfillment workflow
      processPurchase(subscriber);
      break;
  }
  
  res.status(200).send('OK');
});

app.listen(3000, () => console.log('Webhook server running'));

E-commerce Integrations

Shopify Integration

ConvertKit's Shopify integration enables e-commerce email marketing:

  • Customer Sync: New Shopify customers become ConvertKit subscribers
  • Purchase Tags: Auto-tag customers based on products purchased
  • Abandoned Cart: Trigger email sequences for abandoned carts
  • Revenue Tracking: See email-attributed sales in ConvertKit

Setup Steps

  1. In ConvertKit, go to Settings → Integrations → Shopify
  2. Click "Connect Shopify"
  3. Log in to your Shopify store
  4. Authorize the connection
  5. Configure: which tags to apply, which sequence to trigger
  6. Enable customer sync

WooCommerce Integration

For WordPress/WooCommerce stores:

  1. Install the ConvertKit plugin from WordPress.org
  2. Go to WooCommerce → Settings → ConvertKit
  3. Enter your API Secret
  4. Configure customer sync settings
  5. Select which tags to apply for purchases

Stripe Integration

Connect Stripe to ConvertKit for payment-triggered automation:

  • New customers tagged as "customer"
  • Subscription started → add to onboarding sequence
  • Payment failed → trigger retry sequence
  • Subscription canceled → move to re-engagement sequence

Frequently Asked Questions

Does ConvertKit integrate with Salesforce?

Not directly. Use Zapier or Make to connect ConvertKit with Salesforce. Create Zaps to sync new subscribers, add tags based on CRM fields, and trigger Salesforce tasks from ConvertKit events.

Can I use ConvertKit with HubSpot?

Yes, via Zapier. Common workflows: HubSpot form submission → ConvertKit subscriber, ConvertKit tag added → HubSpot contact property updated, ConvertKit purchase → HubSpot deal created.

How do I sync ConvertKit with Google Sheets?

Use Zapier: 1) Create a Zap, 2) Trigger: ConvertKit New Subscriber, 3) Action: Google Sheets Create Spreadsheet Row, 4) Map subscriber data to columns. New subscribers will automatically appear in your spreadsheet.

Does ConvertKit integrate with payment processors?

Yes, ConvertKit has native Stripe integration. For PayPal, use Zapier or manual integration via webhook. For other processors (Square, Authorize.net), use Zapier.

Can I embed ConvertKit forms on any website?

Yes. ConvertKit provides embed code (JavaScript or inline frame) that works on any website. You can also use the HTML form action endpoint for server-side submissions.
