Como Configurar o Seu Trezor

Passo a Passo para Iniciantes

  1. Descarregue o Trezor Suite: Visite trezor.io/start e descarregue a aplicação oficial
  2. Ligue o dispositivo: Use o cabo USB incluído para ligar ao computador
  3. Instale o firmware: Na primeira utilização, instale o firmware mais recente
  4. Crie uma nova carteira: Escolha "Criar nova carteira" (não recuperar)
  5. Guarde a seed phrase: Escreva as 12-24 palavras em papel
  6. Confirme a seed: Verifique as palavras no dispositivo
  7. Configure o PIN: Escolha um PIN seguro
  8. Pronto! Agora pode receber criptomoedas

Receber Criptomoedas no Trezor

Para transferir criptomoedas de uma exchange (como Binance ou Bybit) para o seu Trezor:

  1. Abra o Trezor Suite
  2. Selecione a criptomoeda desejada
  3. Clique em "Receber"
  4. Confirme o endereço no ecrã do dispositivo
  5. Copie o endereço e use-o na exchange para enviar
