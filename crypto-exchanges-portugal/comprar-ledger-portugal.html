Como Configurar o Seu Ledger

Passo a Passo para Iniciantes

  1. Descarregue o Ledger Live: Visite ledger.com/start e descarregue a aplicação oficial para o seu sistema operativo
  2. Ligue o dispositivo: Use o cabo USB incluído para ligar ao computador
  3. Configure como novo: Escolha "Configure como novo dispositivo" (não restaurar)
  4. Escolha o PIN: Defina um PIN seguro de 4-8 dígitos
  5. Guarde a seed phrase: Escreva as 24 palavras em papel (fornecido)
  6. Confirme a seed: Verifique as palavras no ecrã do dispositivo
  7. Instale as apps: Adicione as criptomoedas que pretende usar
  8. Pronto! Agora pode receber criptomoedas no seu Ledger

Receber Criptomoedas no Ledger

Para transferir criptomoedas de uma exchange (como Binance ou Bybit) para o seu Ledger:

  1. Abra o Ledger Live
  2. Selecione a criptomoeda desejada
  3. Clique em "Receber"
  4. Confirme o endereço no ecrã do dispositivo
  5. Copie o endereço e use-o na exchange para enviar
