Critical Security Tips

⚠️ NEVER Store in Password Manager:
  • Complete seed phrases (all 24 words in one place)
  • Private keys
  • Passphrase for hardware wallet

Write these on paper and store securely offline.

Best Practice for Seed Phrases

  1. Write seed phrase on paper
  2. Store in multiple secure locations
  3. Never store digitally (including password managers)
  4. Use password manager for exchange passwords only

What You CAN Store

  • Exchange passwords
  • Email passwords
  • Notes about which exchanges you use
  • 2FA backup codes (in secure notes)

How to Set Up

Step 1: Create Account

  1. Choose password manager
  2. Create account with strong master password
  3. Save master password securely (not digitally)

Step 2: Import Existing Passwords

  1. Most managers can import from browsers
  2. Review and delete old passwords

Step 3: Generate New Passwords

  1. For each crypto exchange, generate new password
  2. Minimum 20 characters
  3. Include letters, numbers, symbols

Step 4: Enable 2FA

  1. Enable 2FA on password manager
  2. Use authenticator app (not SMS)
