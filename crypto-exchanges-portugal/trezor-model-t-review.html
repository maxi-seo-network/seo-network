Setup Guide

Initial Setup

  1. Unbox and connect via USB-C
  2. Go to trezor.io/start
  3. Install Trezor Suite
  4. Follow setup wizard
  5. Create PIN on device
  6. Write recovery seed
⚠️ Critical: Write recovery seed on paper, never digitally. Anyone with these words can steal your crypto.
