Secure Element Protection

For the first time in Trezor's history, the Safe 5 incorporates a certified Secure Element – an EAL6+ rated chip that provides hardware-isolated security. This tamper-resistant component:

  • Stores your PIN and protects against brute-force attacks
  • Safeguards private keys with hardware-level encryption
  • Resists physical extraction attempts
  • Provides an additional security layer without compromising open-source principles

The inclusion of a Secure Element addresses a long-standing debate in the cryptocurrency community. While some purists argued against closed-source chips, Trezor has implemented the Secure Element in a way that maintains full firmware transparency. The chip handles only specific security functions while the main processor runs fully auditable open-source code.

Shamir Backup

The Trezor Safe 5 supports Shamir Backup, an advanced recovery method based on Shamir's Secret Sharing algorithm. Instead of a single 12 or 24-word recovery seed, Shamir Backup splits your wallet's master secret into multiple shares:

  • You can create 1-16 shares total
  • Set a threshold (e.g., require 3 of 5 shares to recover)
  • Shares can be distributed to trusted individuals or secure locations
  • No single point of failure – lose some shares without losing access

This system offers far superior protection compared to traditional seed phrases. Even if someone finds one of your shares, they cannot access your wallet without the required threshold. This makes Shamir Backup ideal for large holdings or inheritance planning.
