What Is Ledger Nano X?

Ledger Nano X is a hardware wallet that stores cryptocurrency private keys offline. This makes it virtually immune to online hacks.

Setup Guide

Initial Setup

  1. Unbox and verify seal is intact
  2. Connect via USB or Bluetooth
  3. Follow on-screen setup
  4. Write down 24-word recovery phrase
  5. Store phrase in safe place
⚠️ Critical: Never share your recovery phrase. Anyone with these 24 words can steal your crypto.

Installing Apps

  1. Open Ledger Live on computer
  2. Connect Ledger Nano X
  3. Go to Manager
  4. Install apps for coins you hold
  5. Add accounts to see balances

Ledger Live App

Ledger Live is the companion app for managing your crypto:

  • View balances and portfolio
  • Send and receive crypto
  • Stake certain coins
  • Buy crypto directly (via partners)
  • Swap tokens

Using with DeFi

Ledger Nano X integrates with:

  • MetaMask (for Ethereum DeFi)
  • ParaSwap (DEX aggregator)
  • Lido (staking)
  • 1inch (DEX)

How to Connect to MetaMask

  1. Install Ethereum app on Ledger
  2. Open MetaMask in browser
  3. Click connect hardware wallet
  4. Select Ledger
  5. Approve on device
