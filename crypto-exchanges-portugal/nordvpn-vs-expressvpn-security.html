Security Features Comparison

Encryption Standards

Both NordVPN and ExpressVPN use military-grade encryption that is virtually unbreakable with current technology. However, there are technical differences worth understanding.

NordVPN Encryption

  • Algorithm: AES-256-GCM (Galois/Counter Mode)
  • Key Exchange: 4096-bit RSA
  • Protocols: NordLynx (WireGuard), OpenVPN (UDP/TCP), IKEv2/IPsec
  • Perfect Forward Secrecy: Yes

NordVPN's use of AES-256-GCM is modern and efficient. The GCM mode provides authenticated encryption, meaning it verifies data integrity while encrypting. This is faster than older modes while maintaining the same security level.

ExpressVPN Encryption

  • Algorithm: AES-256-CBC (Cipher Block Chaining)
  • Key Exchange: 4096-bit RSA
  • Protocols: Lightway, OpenVPN (UDP/TCP), IKEv2
  • Perfect Forward Secrecy: Yes

ExpressVPN uses AES-256-CBC, which is slightly older but still completely secure. CBC mode is widely trusted and has been the industry standard for years. ExpressVPN's proprietary Lightway protocol is optimized for speed and reliability.

Winner: Both offer excellent encryption. NordVPN's GCM mode is technically more modern, but the practical security difference is negligible for users.

No-Logs Policy

A no-logs policy means the VPN provider doesn't record, store, or share your online activities. This is critical for privacy—without logs, even if authorities request data, there's nothing to hand over.

NordVPN No-Logs Policy

NordVPN's no-logs policy has been independently audited multiple times:

  • First audit (2018): By PwC Switzerland
  • Second audit (2020): By Deloitte
  • Third audit (2022): Comprehensive verification by Deloitte

The audits confirmed NordVPN doesn't log IP addresses, connection timestamps, session information, bandwidth usage, or traffic data. They only collect minimal data for service functionality: email for account, payment information, and crash reports (optional).

ExpressVPN No-Logs Policy

ExpressVPN has also undergone independent audits:

  • Cure53 audit (2019): Verified security practices
  • PwC audit (2020): Confirmed no-logs claims

ExpressVPN states it doesn't log browsing history, traffic destination, DNS queries, or IP addresses. In 2017, ExpressVPN demonstrated its no-logs policy when Turkish authorities seized a server investigating the assassination of Russian ambassador Andrei Karlov—no useful data was found.

Winner: Tie. Both have independently audited no-logs policies with strong track records. NordVPN has more recent comprehensive audits.

Jurisdiction and Privacy Laws

Jurisdiction determines which laws govern a VPN provider and whether they can be compelled to hand over user data. For privacy-conscious users, jurisdiction outside intelligence-sharing alliances is crucial.

NordVPN — Panama

NordVPN operates under Panamanian jurisdiction. Panama is outside the Five Eyes, Nine Eyes, and Fourteen Eyes surveillance alliances. Key benefits:

  • No mandatory data retention laws
  • Government cannot compel data production without court order
  • Panama doesn't participate in international intelligence sharing
  • Strong financial privacy laws

Note: NordVPN is owned by Nord Security, which has offices in Lithuania and the Netherlands. However, the legal entity and data processing remain under Panamanian jurisdiction.

ExpressVPN — British Virgin Islands

ExpressVPN is based in the British Virgin Islands (BVI), a British Overseas Territory. While BVI is technically associated with the UK, it has its own legal system and doesn't participate in Five Eyes data sharing:

  • BVI has no mandatory data retention requirements
  • Local courts must approve any government data requests
  • BVI Companies Act protects corporate privacy
  • Not part of Five Eyes intelligence sharing

Important: ExpressVPN was acquired by Kape Technologies in 2021. Kape is based in the UK (Five Eyes member). While ExpressVPN maintains operational independence and its BVI jurisdiction, some privacy advocates note this ownership structure.

Winner: NordVPN edges ahead due to cleaner jurisdiction. Panama has no ties to Five Eyes countries, while ExpressVPN's parent company is UK-based. However, both jurisdictions are privacy-friendly in practice.

Additional Security Features

Kill Switch

Both VPNs include a kill switch that blocks internet traffic if the VPN connection drops. This prevents data leaks during disconnections. NordVPN offers both system-level and app-level kill switches. ExpressVPN's kill switch is called "Network Lock" and blocks all traffic on connection failure.

RAM-Only Servers

Both providers use RAM-only servers that wipe all data on reboot:

  • NordVPN: All servers run on RAM-only infrastructure
  • ExpressVPN: TrustedServer technology (RAM-only)

This means even if authorities physically seized a server, no data would be recoverable—everything is in volatile memory.

Split Tunneling

Split tunneling lets you choose which apps use the VPN and which connect directly:

  • NordVPN: Available on Windows, macOS, Android, and allows per-app configuration
  • ExpressVPN: Available on all platforms with granular app control

Double VPN / Multi-Hop

Only NordVPN offers double VPN (routing through two servers) for extra security. ExpressVPN doesn't provide this feature. For users requiring maximum anonymity, NordVPN's Double VPN adds an extra encryption layer.

Onion Over VPN

NordVPN offers Onion Over VPN, combining VPN encryption with Tor anonymity. ExpressVPN doesn't have this built-in, though you can manually configure Tor over VPN.

Winner: NordVPN for additional security features (Double VPN, Onion Over VPN). ExpressVPN matches core security features.

Speed and Performance

Security matters, but a slow VPN is unusable. Both providers have developed proprietary protocols for speed.

NordVPN Speed — NordLynx Protocol

NordVPN's NordLynx protocol is built around WireGuard, a modern VPN protocol known for speed:

  • WireGuard uses less code than OpenVPN (4,000 vs 100,000+ lines)
  • Faster handshake and connection times
  • Lower battery usage on mobile devices
  • Consistently top-ranked in independent speed tests

In independent benchmarks, NordLynx typically achieves 90-95% of base internet speed on nearby servers and 70-85% on distant servers.

ExpressVPN Speed — Lightway Protocol

ExpressVPN developed Lightway as its proprietary protocol:

  • Lightweight codebase (~1,000 lines)
  • Faster connection times than OpenVPN
  • Designed for stability during network switching
  • Audited by Cure53 for security

Lightway is fast but typically trails NordLynx in benchmarks by 5-15%.
